## Running the script

:::danger[Warning]
Script must be run as **root**, and also on the server where the Remnawave node will be installed.
:::

Enter the command on your server:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/install_remnawave.sh)
```

## Installing the node

<Steps>

1. Select `Install Remnawave components`

    ![Main menu RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/en/main_menu.webp)

2. Then select `Install only the node`

    ![Install node RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/en/install_node.webp)

3. Log in to the control panel, go to the `Nodes` → `Management` section, select the desired node, and click the `Important Information` button. In the pop-up window, you will see an icon to copy the certificate — click on it.

    ![SSL_CERT RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/en/ssl_cert.webp)

4. Script will ask you for the selfsteal domain that you specified during the panel installation or through the [`Add node to panel`](https://wiki.egam.es/installation/add-node/) section, the panel's IP address, and the SSL certificate from the panel.

    :::danger[Warning]
    Don't forget to add a DNS record for your domain.
    :::

    ![Menu node end RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/en/menu_node_end.webp)

5. Then the script will start the node and connect it to the panel.

    ![Menu node end2 RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/en/menu_node_end2.webp)

</Steps>

## Updating the script

Make sure to update the script before running things in it. You can do this by selecting `Check for updates script` in the script menu or by viewing the GitHub repository for updates.
