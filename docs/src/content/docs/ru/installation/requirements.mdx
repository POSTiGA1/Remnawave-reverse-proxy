## Remnawave Panel

### Системные требования

- **OS**: Рекомендуется **Ubuntu** или **Debian**.
- **RAM**: Минимум 2GB, рекомендуется 4GB.
- **CPU**: Минимум 2 ядра, рекомендуется 4 ядра.
- **Storage**: Минимум и рекомендуется 20GB.

## Remnawave Node

### Системные требования

- **OS**: Рекомендуется **Ubuntu** или **Debian**.
- **RAM**: Минимум 1GB.
- **CPU**: Минимум 1 ядро.

:::tip
`Remnawave` может работать с меньшими системными требованиями, но это не рекомендуется. Некоторые фоновые процессы могут быть ресурсоемкими.

`Remnawave Node` сам по себе не требует много ресурсов, но Xray-core может потреблять много CPU и RAM под нагрузкой.
:::

## Настройка домена
Скрипт поддерживает два метода настройки домена: через [Cloudflare](https://dash.cloudflare.com/) или с использованием ACME с вашим хостинг-провайдером.

### DNS записи для панели и ноды на одном сервере

| Type  | Name              | Content          | Proxy status  |
| ----- | ----------------- | ---------------- | ------------- |
| A     | example.com       | your_server_ip   | DNS only      |
| CNAME | panel.example.com | example.com      | DNS only      |
| CNAME | sub.example.com   | example.com      | DNS only      |
| CNAME | node.example.com  | example.com      | DNS only      |

:::tip
Запись node.example.com не является обязательной для selfsteal ноды — вы также можете использовать example.com для selfsteal.
:::

### DNS записи для панели и ноды на разных серверах

| Type  | Name              | Content                 | Proxy status  |
| ----- | ----------------- | ----------------------- | ------------- |
| A     | example.com       | your_server_ip          | DNS only      |
| CNAME | panel.example.com | example.com             | DNS only      |
| CNAME | sub.example.com   | example.com             | DNS only      |
| A     | node.example.com  | your_server_ip_vps_node | DNS only      |
