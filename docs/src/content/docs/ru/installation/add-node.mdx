## Установка скрипта

:::danger[Важно]
Скрипт нужно запускать с **правами root**, а так же на сервере, на котором установлена панель Remnawave.
:::

Введите данную команду на вашем сервере, где установлена панель Remnawave:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/install_remnawave.sh)
```

## Добавить ноду в панель

<Steps>

1. Выберите пункт `Установка компонентов Remnawave`

    ![Main menu RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/main_menu.webp)

2. Затем выберите пункт `Добавить ноду в панель`

    ![Add node to panel RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/add_node_to_panel.webp)

3. Введите ваш новый домен для `selfsteal` который будет использоваться для подключения ноды к панели, а так же название для самой ноды.

    :::danger[Важно]
    Не забудьте добавить DNS-запись для вашего домена.
    :::
    
    ![Add node to panel2 RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/add_node_to_panel2.webp)

4. После успешного добавления ноды в панель, запустите скрипт на вашем новом сервере, где будет установлена нода Remnawave, выберите пункт `Установка компонентов Remnawave`, а затем [`Установить только ноду`](https://wiki.egam.es/ru/installation/node-only/).

</Steps>

## Обновление скрипта

Убедитесь, что вы обновили скрипт перед выполнением каких-либо действий в нем. Вы можете сделать это, выбрав `Проверить обновления скрипта` в меню скрипта или просмотрев репозиторий GitHub на наличие обновлений.
