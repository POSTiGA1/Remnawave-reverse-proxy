## Установка скрипта

:::danger[Внимание]
Скрипт нужно запускать с правами root.
:::

Введите данную команду на вашем сервере:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/install_remnawave.sh)
```

## Установка панели и ноды
:::danger[Внимание]
Этот метод установки не рекомендуется для производственных сред, так как он даже не документирован в документации Remnawave.  
Для лучшего опыта, пожалуйста, используйте методы установки `Только панель` и `Только нода`, в зависимости от того, хотите ли вы установить панель или ноду.
:::

Выберите пункт "Установка компонентов Remnawave"

![Main menu RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/main_menu.ru.png)

Затем выберите пункт "Установить панель и ноду на один сервер"

![Install panel and node RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/install_panel_node.ru.png)

После завершения процесса скрипт автоматически перезапустит панель и предоставит все необходимые данные для входа в неё.

![Menu end RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/menu_end_ru.png)

:::danger[Сохраните ваши учетные данные]
После завершения установки скрипт отобразит уникальный URL-адрес для входа в панель управления.  
**Обязательно сохраните этот URL-адрес.** Он содержит секретный ключ, необходимый для доступа.

Пример ссылки для входа:
```
https://p.example.com/auth/login?SECRET_KEY=SECRET_KEY
```
Без этого URL-адреса **вы не сможете** войти в панель.
:::
