## Установка скрипта

:::danger[Внимание]
Скрипт нужно запускать с правами root.
:::

Введите данную команду на вашем сервере:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/install_remnawave.sh)
```

## Установка ноды

<Steps>

1. Выберите пункт "Установка компонентов Remnawave"

    ![Main menu RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/main_menu.ru.png)

2. Затем выберите пункт "Установить только ноду"

    ![Install node RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/install_node.ru.png)

3. Скрипт запросит у Вас домен selfsteal, который вы указали при установке панели или через пункт "Добавить ноду в панель", ip-адрес панели и сертификат SSL с панели.

    ![Menu node end RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/menu_node_end.ru.png)

4. Зайдите в панель управления, перейдите в раздел "Ноды" → "Управление", выберите нужную ноду и нажмите на кнопку "Важная информация". В появившемся окне вы увидите иконку для копирования сертификата (SSL_CERT) — нажмите на неё.

    ![SSL_CERT RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ssl_cert.ru.png)

5. Затем скрипт запустит ноду и подключит её к панели.

    ![Menu node end2 RU](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/menu_node_end2.ru.png)

</Steps>
