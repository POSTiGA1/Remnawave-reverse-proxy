Netbird — это open-source платформа на базе WireGuard, которая позволяет легко создавать защищённые частные сети. В этом руководстве мы разберём, как установить и настроить Netbird для связывания двух серверов, чтобы они могли безопасно взаимодействовать друг с другом.

## Регистрация в Netbird

<Steps>

1. Откройте ваш браузер и перейдите по ссылке.

    ```bash
    https://app.netbird.io/
    ```

2. Создание учётной записи

    <div style={{ display: 'flex', justifyContent: 'center' }}>
      <img 
        src="https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/netbird/register.png" 
        alt="Netbird registration" 
        style={{ maxWidth: '60%' }} 
      />
    </div>

3. Нажмите `Sign Up`. Выберите способ регистрации: через email, Google, GitHub или Microsoft.

4. Заполните поля (email, пароль, имя) и подтвердите регистрацию через письмо.

5. После этого вы попадёте в Netbird Dashboard.

    ![Netbird peer](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/netbird/peer.png)

</Steps>

## Создание и настройка Setup Key

<Steps>

1. Переходим в раздел Setup Keys. Нажмите кнопку Create Setup Key.

    ![Netbird setup key](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/netbird/setup_key.png)

2. Создание ключа.

    <div style={{ display: 'flex', justifyContent: 'center' }}>
      <img 
        src="https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/netbird/create_new_setup_key.png" 
        alt="Netbird create setup key" 
        style={{ maxWidth: '60%' }} 
      />
    </div>

3. Введите имя для ключа (например, RemnawaveKey).

4. Включите ползунок `Make this key reusable`, чтобы ключ можно было использовать для нескольких серверов (до 30 peers).

5. Установите `Expires in` на `Unlimited`, чтобы ключ не истекал.

6. Нажмите `Create Setup Key`.

7. Сохраните ваш ключ.

    <div style={{ display: 'flex', justifyContent: 'center' }}>
      <img 
        src="https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/netbird/key.png" 
        alt="Netbird key" 
        style={{ maxWidth: '60%' }} 
      />
    </div>

    Появится окно с сгенерированным ключом (например, 12B653C1-4E03-452E-9DA9-40523493205B).
    Скопируйте ключ и сохраните его в безопасном месте для дальнейшего использования.

</Steps>

## Установка Netbird на сервер панели

<Steps>

1. Нажмите кнопку Install Netbird.
    Откроется окно с инструкцией по установке.

    <div style={{ display: 'flex', justifyContent: 'center' }}>
      <img 
        src="https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/netbird/install_netbird.png" 
        alt="Netbird create setup key" 
        style={{ maxWidth: '75%' }} 
      />
    </div>

2. Установите Netbird на сервер панели Remnawave. Выполните команду для установки Netbird:

    ```bash
    curl -fsSL https://pkgs.netbird.io/install.sh | sh
    ```

    ![Netbird setup](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/netbird/setup_netbird.png)

3. Запуск Netbird с ключом. Выполните команду для подключения с использованием сохранённого ключа:

    ```bash
    netbird up --setup-key AE53280D-88CC-4250-9228-401AB1C7C041
    ```

    ![Netbird setup](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/netbird/setup_netbird2.png)

4. Дождитесь сообщения `Connected`, указывающего на успешное подключение.

</Steps>

Проделайте те же действия установки, что и на сервере панели.

Вернитесь в Dashboard и перейдите в раздел Peers. Убедитесь, что peer ноды отображается как подключённый. 

![Netbird peer](https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/netbird/peer_netbird_node.png)

## Настройка брандмауэра (UFW) на сервере с нодой

Посмотрите в Dashboard IP-адрес Netbird панели (в нашем случае это 100.88.38.121, у вас может быть другой).
Выполните команду для открытия порта 2222 (TCP) для этого IP:

```bash
ufw allow from 100.88.38.121 to any port 2222 proto tcp
```

Перезагрузите брандмауэр:

```bash
ufw reload
```

## Настройка Netbird в панели Remnawave

<Steps>

1. Перейдите в вашу панель Remnawave, зайдите в раздел `Ноды` → `Управление`.

2. Выберите ноду, где установлен Netbird.

3. В поле `Адрес` введите IP-адрес Netbird ноды (в нашем случае это 100.88.74.220, у вас может быть другой).

    <div style={{ display: 'flex', justifyContent: 'center' }}>
      <img 
        src="https://raw.githubusercontent.com/eGamesAPI/remnawave-reverse-proxy/refs/heads/main/docs/src/assets/ru/netbird/ip_netbird_node.png" 
        alt="Netbird create setup key" 
        style={{ maxWidth: '75%' }} 
      />
    </div>
    
    Нажмите кнопку `Сохранить`.

</Steps>

## Итог

- Теперь ваша панель Remnawave и сервер с нодой связаны через защищённый туннель Netbird. 
- Вы можете использовать этот канал для безопасного обмена данными или управления нодой через порт 2222. 
- Netbird упрощает настройку и обеспечивает высокую безопасность благодаря WireGuard.
